Currency Converter V 0.1.2, Welmo Menu
The menu is composed of three options
  • The Synchronization, which launch the synchronization with the selected synch source (version 0.2 support only BCE synch)
  • The configuration,which allow the configuration of currency parameter:
    • Default Source currency
    • Default amount value for source currency
    • Synch source (dynamic source selection not yet on 0.2, fixed to BCE)
    • List of currency to be shown in the currency list
